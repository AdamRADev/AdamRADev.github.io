Apa Itu Lunassynergy?

Lunassynergy Adalah Server Minecraft Yang Bertemakan RPG, Server Ini Bisa Dimainkan Di MCPE & MCPC, Menariknya Dalam Server Ini, Terdapat Fitur-Fitur RPG Seperti Dungeon, Sword RPG, Guild, Boss. Server Ini Dapat Dimainkan Pada Versi 1.21x - 1.8x. Kelebihan Server Ini: 

Player Ramai & Baik

Diserver Ini Playernya Ramah Dan Baik, Dan Disini Kamu Bisa Berteman Dengan Orang Lain. Membuat Base Bersama Dan Mengalahkan Boss! 

Selalu Update & Support Semua Platform

Diserver Ini, Akan Selalu Update Seiringan Season Berjalan. Semua Orang Bisa Join Dari Minecraft Pocket Edition & Minecraft Java Edition.

Admin Fast Respond & Baik

Didalam Server Ini Terdapat Admin Yang Fast Respond Kepada Semua Player, Dan Admin Diserver Ini Ramah & Baik. 

Cara Join Diserver Ini

Inilah Tutorial Join Kedalam Server Lunassynergy Di Minecraft Java Edition.

1

Tambahkan Server

Pergi Ke Menu "Multiplayer Menu", Lalu Tekan "Add Server".
2

Masukkan Nama & IP Address Server

Masukkan Lunassynergy Dibagian Nama, Lalu Masukkan Juga play.lunassynergy.my.id:19132 Dibagian IP Address, Lalu Tekan "Done".
3

Sudah Selesai!

Cerita Awal Dari Lunassynergy

Lore dari awal: Lunassynergy itu dibentuk dari 2 komunitas, yaitu Lunarian & Sussynergy. Awalnya Sussynergy bikin server minecraft crossplay biasa, ga ada plugin yang gimana-gimana. Lalu salah satu member Sussynergy itu mengundang salah satu admin Lunarian buat diajak main bareng di server Sussynergy. Sekitar beberapa hari berlalu sampai akhirnya servernya kena hack sama hacker 2b2t. Setelah itu Sussynergy bikin server lagi dan member member Lunarian mulai pada berdatangan 1 per 1. Sayangnya karena servernya sering crash waktu itu akhirnya admin Sussynergy & Lunarian nyusun rencana buat collab bikin server baru. Lore mulai masuk Lunassynergy: Sekitar kurang lebih 1 bulan akhirnya Lunarian & Sussynergy collab, jadilah Lunassynergy. Lalu server minecraft Lunassynergy season 1 pun dibuat untuk menampung 2 komunitas tersebut. Sayangnya, belum ada seminggu servernya terkena hack lagi yang membuat servernya terpaksa harus masuk ke season 2
